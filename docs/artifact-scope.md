# Artifact Scope Statement

This repository is a research artifact designed to support reproducible generation of synthetic multi-turn conversations for testing workflows.

## In Scope

- Scenario generation grounded in policy text and anchors.
- Persona-conditioned multi-turn dialogue simulation.
- Edge-case scoring and iterative scenario improvement.
- UI and CLI entry points for running and inspecting outputs.

## Out of Scope (Current Release)

- Production hardening and SLA-oriented operations.
- Full legal validation of third-party non-code assets.
- End-to-end benchmark automation for all paper experiments.
- Complete implementation of convergence-based stopping criteria (DP6).
